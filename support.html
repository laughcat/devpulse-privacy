❓ Frequently Asked Questions

Q: How do I connect RevenueCat?
A: Open the app → Add a project → Select RevenueCat → Enter your v2 API Key. Keys are stored in your device Keychain and never leave your device.

Q: How do I connect App Store Connect?
A: Generate a .p8 API Key in App Store Connect, then select it in DevPulse. The app generates JWT signatures locally using CryptoKit — your key never leaves your device.

Q: What's the difference between Free and Pro?
A: Free supports 1 project with Small & Medium widgets and basic metrics. Pro unlocks unlimited projects, Large widgets, anomaly detection alerts, and daily revenue summaries.

Q: Is my data secure?
A: Yes. 100% client-side architecture — no backend servers. API Keys are stored in iOS Keychain with hardware-level encryption. All data stays on your device. See our Privacy Policy for details.

Q: How do I cancel my subscription?
A: iOS Settings → Your Apple ID → Subscriptions → DevPulse → Cancel. All subscriptions are managed by Apple.

Q: Which data sources are supported?
A: RevenueCat, App Store Connect, and GitHub (Stars & Issues).

❓ 常见问题

Q: 如何连接 RevenueCat？
A: 打开 App → 添加项目 → 选择 RevenueCat → 输入你的 v2 API Key。Key 存储在设备 Keychain 中，不会上传。

Q: 如何连接 App Store Connect？
A: 在 App Store Connect 中生成 .p8 API Key，然后在 DevPulse 中选择该文件。App 使用 CryptoKit 在本机生成 JWT 签名，密钥不离开设备。

Q: Free 和 Pro 有什么区别？
A: Free 版支持 1 个项目、Small / Medium Widget 和基础指标。Pro 版解锁无限项目、Large Widget、异常检测通知和每日收入摘要。具体价格见 App Store 产品页面。

Q: 我的数据安全吗？
A: 安全。100% 纯本地架构，无后端服务器。API Key 存储在 iOS Keychain 中（硬件级 AES 加密），所有数据都在你的设备上。详见 隐私政策。

Q: 如何取消订阅？
A: iOS 系统设置 → 你的 Apple ID → 订阅 → DevPulse → 取消订阅。所有订阅由 Apple 管理。

Q: 支持哪些数据源？
A: RevenueCat、App Store Connect、GitHub（Stars / Issues）。
