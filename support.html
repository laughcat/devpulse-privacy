❓ Frequently Asked Questions

Q: 如何连接 RevenueCat？
A: 打开 App → 添加项目 → 选择 RevenueCat → 输入你的 v2 API Key。Key 存储在设备 Keychain 中，不会上传。

Q: 如何连接 App Store Connect？
A: 需要在 App Store Connect 生成一个 .p8 API Key，然后在 DevPulse 中选择该文件。App 使用 CryptoKit 在本机生成 JWT 签名，密钥不离开设备。

Q: Free 和 Pro 有什么区别？
A: Free 版支持 1 个项目 + Small/Medium Widget。Pro 版（$4.99/月 或 $39/年）支持无限项目 + Large Widget + 异常检测通知 + 每日摘要。

Q: 我的数据安全吗？
A: 100% 本地架构，无后端服务器。API Key 存储在 iOS Keychain（硬件级加密），所有数据都在你的设备上。详见 隐私政策。

Q: 如何取消订阅？
A: iOS 系统设置 → 你的 Apple ID → 订阅 → DevPulse → 取消订阅。所有订阅由 Apple 管理。

Q: 支持哪些数据源？
A: RevenueCat + App Store Connect + GitHub（Stars / Issues）。Stripe 将在未来版本中支持。
