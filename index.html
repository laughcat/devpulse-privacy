Privacy Policy

Last updated: June 21, 2026

DevPulse (hereinafter "we") is an application that runs entirely on your device.
The core principle of this Privacy Policy is: we do not collect any of your data.

1. What We Don't Collect

  • No account registration required — no name, email, or any personal information needed
  • No usage tracking — no analytics, no instrumentation, no data reporting
  • No device information collection — no IDFA access, no device identifiers
  • No data uploaded to servers — we have no servers

2. Where Data Is Stored

All your data is stored entirely on your device:

  • API Keys, secrets → iOS Keychain (hardware-level AES encryption)
  • Project configuration, cached data → App local sandbox + App Group shared container
  • No data ever leaves your device

3. Third-Party API Calls

The app makes HTTPS requests directly to the following platforms using the API Keys you provide:

  • RevenueCat API — fetch subscription revenue data
  • App Store Connect API — fetch download and rating data
  • GitHub API — fetch repository data

These requests are sent directly from your device to third-party servers, without passing through any intermediate servers.

4. Data Deletion

All data is stored locally on your device. You can delete it within the app:

  • Go to Settings → Data Management → tap "Delete All Data"

Since we cannot remotely access your device, we cannot delete data on your behalf. Deletion is entirely under your control.

5. Children's Privacy

This service is not intended for children under the age of 13.

6. Privacy Policy Updates

We may update this Privacy Policy from time to time. Updated policies will be published within the app.

7. Contact Us

If you have any questions, please contact us via:

  • Email: laughcat.lee@outlook.com
  • In-app feedback: Settings → Send Feedback

本隐私政策的核心原则是：我们不收集您的任何数据。

1. 我们不收集什么

2. 数据存储位置

您的所有数据完全存储在您的设备上：

App 通过您提供的 API Key 直接向以下平台发起 HTTPS 请求：

这些请求从您的设备直接发送到第三方服务器，不经由任何中间服务器。

4. 数据删除

所有数据存储在您的设备本地。您可以在 App 内完成删除：

由于我们无法远程访问您的设备，我们无法代为删除。删除操作由您完全控制。
